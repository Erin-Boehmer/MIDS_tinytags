TinyTags is a final project created by

for the Scaling Up! Really Big Data course in the UC Berkeley Master of Information and Data Science program. Check out the source code on Github!

TinyTags allows you to easily search the 80 million tiny images dataset provided by MIT and NYU using a set of predefined tags and derived accuracy scores. These tags are obtained from the smaller 60,000 CIFAR-10 dataset that provides 32x32 color images labeled as "deer", "automobile", "truck", "cat", "dog", "bird", "airplane", "ship", "frog", and "horse".

In order to build TinyTags, we trained a distributed Random Forests machine learning algorithm using Spark MLlib using the CIFAR-10 data. The 230 GB of tiny images data, stored in HDFS, was then passed through our classifier to derive the most-likely tag prediction for each image. We then used Solr to index the imagery data so that the tagged images could be made searchable through the TinyTags webapp.

We hope you enjoy using TinyTags! For a more detailed description of the TinyTags project, please check out our white paper.
